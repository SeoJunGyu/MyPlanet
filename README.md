# 🎮 MyPlanet

다양한 선택지를 통해 매 순간마다 새로운 전략으로 플레이하는 로그라이트 뱀서 게임

---

## 📌 프로젝트 소개

* 개발 인원 : 개발자 3인, 기획자 3인 (협업 프로젝트)
* 개발 기간 : 2025.11.07 ~ 2026.01.06 (8주)
* 빌드 : Android (PlayStore)
* 개발 환경 : Unity 6.x

---

### 🎯 게임 설명

* 모든 웨이브를 버티면서 마지막 보스를 잡아 스테이지를 클리어합니다.

## ⚙️ 주요 기능 및 시스템

### Tower / Planet (Player)

* 타원기반 행성 이동 시스템
* 공격타워, 버프타워 등 다양한 타워 시스템
* 타워 강화 시스템

### Enemy

* 다양한 패턴 기반 
* 플레이어 추적 및 공격 로직

### Game Systems

* 
* 증강(선택지) 시스템
* UI 시스템
